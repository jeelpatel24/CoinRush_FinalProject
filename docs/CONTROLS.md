# Controls

This file describes the default controls used in CoinRush.

## Keyboard (PC)
- Move left: A or Left Arrow
- Move right: D or Right Arrow
- Jump: Space or W or Up Arrow
- Pause: Esc
- Restart: R

## Gamepad (if supported)
- Left stick / D-Pad: Move
- A / X button: Jump
- Start / Options: Pause
